# 求职全流程模拟 —— 三个方向

> 方向：海外品牌增长/运营 | 互联网产品运营/数据运营 | 快消/新能源海外市场实习生
> 工具链：boss-agent-cli + OpenCLI + AI Skills
> 数据根目录：`D:\AI_code_test\pipeline\`

---

## Phase 1: 搜索职位（boss-agent-cli）

### 方向 A：海外品牌增长/运营

```bash
# 搜索职位
boss search "海外品牌增长" --city 上海 --save results.json
boss search "出海运营" --city 深圳 --save results.json
boss search "品牌出海" --city 广州 --save results.json

# 福利过滤（只看双休、五险一金）
boss search "海外增长" --welfare "双休,五险一金" --save results.json

# 加入候选池
boss shortlist add <security_id> <job_id>
boss shortlist list
```

**预期结果：** 获取 3 个城市 × 多个关键词 ≈ 50-100 条职位，筛选后候选池保留 10-15 个

**文件：** `pipeline/职位池/海外品牌增长/search_results.json`

### 方向 B：互联网产品运营/数据运营

```bash
# 搜索职位
boss search "产品运营" --city 北京 --save results.json
boss search "数据运营" --city 上海 --save results.json

# 按薪资过滤
boss search "数据运营" --salary "15k-30k" --save results.json

# 查看详情
boss detail <security_id> > pipeline/职位池/互联网运营/detail_字节跳动.txt
boss detail <security_id> > pipeline/职位池/互联网运营/detail_腾讯.txt
```

**预期结果：** 获取 2 个城市 × 2-3 个关键词 ≈ 40-80 条职位

### 方向 C：快消/新能源海外市场实习生

```bash
# 搜索职位
boss search "快消 海外市场 实习" --city 上海 --save results.json
boss search "新能源 海外市场 实习" --city 上海 --save results.json
boss search "出海 实习生" --city 深圳 --save results.json

# 加入候选池
boss shortlist add <security_id> <job_id>
boss shortlist list
```

**预期结果：** 实习生岗位相对少，预估 10-20 条

## Phase 2: 面经调研（OpenCLI + 多平台）

### 从知乎获取面经

```bash
# 搜索面经
opencli zhihu search "海外运营 面经"
opencli zhihu search "产品运营 面试"
opencli zhihu search "品牌出海 工作体验"

# 保存结果
opencli zhihu search "数据运营 面经" -f json > pipeline/面经/知乎/数据运营.json
```

### 从一亩三分地获取

```bash
opencli 1point3acres search "product operation"
opencli 1point3acres search "marketing intern"
opencli 1point3acres thread <id> > pipeline/面经/一亩三分地/面经详情.txt
```

### 从小红书获取

```bash
opencli xiaohongshu search "互联网运营 面经"
opencli xiaohongshu search "快消 面经"
```

### AI 分析面经

```bash
# 用 boss ai 分析 JD 匹配度
boss ai analyze-jd --file pipeline/职位池/目标职位详情.txt

# AI 润色简历
boss ai optimize --target "海外品牌增长" --resume pipeline/简历/通用版/resume.md
```

## Phase 4: AI 简历优化（boss-agent-cli AI 功能）

### 针对 JD 优化简历

```bash
# Step 1: 分析职位 JD
boss ai analyze-jd --jd pipeline/职位池/目标JD.txt

# Step 2: 针对方向润色简历
boss ai polish --target "海外品牌增长" --resume pipeline/简历/通用版/resume.md > pipeline/简历/品牌增长方向/resume.md

# Step 3: 模拟面试
boss ai interview-prep --jd pipeline/职位池/目标JD.txt > pipeline/面经/面试准备_目标公司.md
```

### 简历版本管理

```
pipeline/简历/
├── 通用版/resume.md              # 基础简历
├── 品牌增长方向/resume.md         # 针对海外增长岗优化
├── 数据运营方向/resume.md         # 针对数据运营岗优化
└── 快消新能源方向/resume.md       # 针对快消新能源岗优化
```

## 完整流程拓扑

```
一天的工作流示例：

上午 10:00   boss search "海外品牌增长" → 发现 5 个新职位
上午 10:15   boss shortlist add → 加入候选池
上午 10:20   boss detail → 看 JD 详情
上午 10:30   boss ai analyze-jd → AI 分析匹配度
上午 10:45   boss ai polish → 针对性优化简历

下午 14:00   opencli zhihu search "海外运营 面经" → 收集面经
下午 14:30   opencli xiaohongshu search "公司名 工作体验"
下午 15:00   opencli maimai search-talents → 找内推人

下午 16:00   boss apply → 投递（需要扫码确认）
下午 16:30   boss pipeline update → 更新状态
下午 17:00   boss stats → 看今日数据
```
